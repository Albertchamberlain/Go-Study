* 以下为常用转义字符🗡

| verb     |          含义           |
|:---------|:----------------------:|
| %d       |        十进制整数        |
| %x,%X    | 大小写方式显示十六进制整数 |
| %o       |        八进制整数        |
| %b       |        二进制整数        |
| %f,%g,%e |         浮点数          |
| %t       |         布尔值          |
| %c       |          字符           |
| %s       |         字符串          |
| %q       |      带双引号字符串      |
| %v       |       内置格式内容       |
| %T       |          类型           |
| %p       |        内存地址         |
| %%       |          字符%          |
| \n       |          换行           |
| \t       |          缩进           |
