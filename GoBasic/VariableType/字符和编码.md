# 字符型概述

* 字符型存放单个字母或单个文字
* Go语言不支持字符类型,在Go语言中所有字符值都转换为对应的编码表中int32值
* Go语言默认使用UTF-8编码


# 字符示例

* 示例

```go
func main() {
	//定义数字
	var i rune=`1 //0x5F20
	fmt.Println(i)

	fmt.Printf("%c\n",i)
	
	//获取转换后的内容
	c:=fmt.Sprintf("%c",i)
	fmt.Println(c)
}
```

* 也可以使用单引号表示一个字符类型,但是本质还是正数

```go
func main() {
	c := '侯'
	fmt.Println(c)      //24352
	fmt.Printf("%T", c) //int32
}
```
